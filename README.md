# RG Lively Gray Theme

A comfortable, eye-friendly gray theme suite for VS Code. Designed to reduce eye strain while maintaining a lively and professional look.

## Features

- **Gray-based Palette**: Carefully selected gray tones to avoid harsh contrast.
- **Eye Comfort**: "Parchment" style text colors and warm/cool variations to suit different lighting conditions.
- **Lively Accents**: Pastel and muted accent colors to keep the code readable and interesting.
- **6 Variants**:
    - **RG Lively Gray (Warm)**: Warm gray background, ideal for long coding sessions.
    - **RG Lively Gray (Cool)**: Blue-gray background, modern and clean.
    - **RG Lively Gray (High Contrast)**: Darker background with high contrast text for maximum clarity.
    - **RG Lively Gray (Warm Light)**: A lighter, brighter version of the Warm theme.
    - **RG Lively Gray (Cool Light)**: A lighter, brighter version of the Cool theme.
    - **RG Lively Gray (HC Light)**: A lighter version of the High Contrast theme.

## Installation

1. Open **Extensions** sidebar in VS Code.
2. Search for `RG Lively Gray Theme`.
3. Click **Install**.
4. Go to **File > Preferences > Color Theme** and select one of the **RG Lively Gray** themes.
